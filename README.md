# FUTURE_FS_01 — Full Stack Developer Portfolio

A professional full-stack developer portfolio built using Next.js, TypeScript, Tailwind CSS, MongoDB, and Framer Motion.

## Features

- Responsive modern UI
- Dynamic project showcase
- Resume download functionality
- Functional contact form
- MongoDB integration
- GitHub and LinkedIn integration
- Custom favicon and branding

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MongoDB Atlas
- Framer Motion
- React
- Node.js

## Featured Projects

### Travel AI – Predictive Intelligence Engine
AI-powered travel analysis dashboard using Google Gemini API.

### Health Insurance Cost Prediction
Machine learning web application for predicting insurance charges.
