# **App Name**: Elite Portfolio

## Core Features:

- Portfolio Section Display: Showcase Hero, About, Skills, Projects, Resume, Contact, and Footer sections with responsive and engaging layouts.
- Project Showcase: Display up to 6 detailed project cards including title, description, technologies, GitHub link, live demo link, and screenshot placeholder.
- Resume and Experience Presentation: Highlight resume summaries, education/experience timelines, and provide a downloadable resume feature.
- Interactive Contact Form: Implement a contact form with fields for name, email, subject, and message, featuring client-side validation and immediate success/error feedback.
- Contact API Backend: Develop a RESTful API using Node.js/Express.js to handle contact form submissions, including server-side validation.
- Secure Contact Data Storage: Persist contact form messages (name, email, subject, message, createdAt) in a MongoDB database using Mongoose.
- SEO Optimization: Implement proper title tags, meta descriptions, and Open Graph tags for improved search engine visibility.

## Style Guidelines:

- Primary Background: A deep, professional charcoal (#1A202C) for a modern dark theme.
- Text Color: Soft white or light gray (#E2E8F0) for excellent readability.
- Accent Color: A sophisticated, muted teal (#4FD1C5) for interactive elements, buttons, and highlights.
- Headings Font: 'Space Grotesk' (sans-serif) for a contemporary, strong, and tech-savvy aesthetic.
- Body Text Font: 'Inter' (sans-serif) ensuring high readability and a clean, professional appearance across all devices.
- Clean, minimalist outlined icons are used consistently throughout the portfolio for navigation, social links, and feature indicators, maintaining a sleek and uncluttered look.
- Fully responsive design for optimal viewing on mobile, tablet, and desktop. Features clear section separation, generous whitespace, and a sticky, professional navigation bar.
- Subtle, smooth animations (powered by Framer Motion) for section transitions, element entrances, and interactive hover effects to enhance user engagement and provide a polished, premium feel.
